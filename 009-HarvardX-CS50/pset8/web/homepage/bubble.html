Bubble sort

Bubble sort is a sorting algorithm that repeatedly sorts the given list compairing each pair of neighboring elements and swapping them if needed. The algorithm stops when no swaps are needed when stepping through the list.

Bubble sort is called so because of smaller elements "bubbling" towards the top of the list during sort process.
